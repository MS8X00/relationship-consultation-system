# Contributing

Thanks for contributing to this skills bundle.

## Scope

This repository is a modular set of relationship consultation skills for AI agents. Keep contributions focused on:

- `skills/*.skill.md` skill quality and clarity
- `docs/` integration and protocol guidance
- `memory-templates/` structure and usability

## How to Contribute

1. Open an issue describing the problem or proposal.
2. Keep each pull request focused on one skill or one documentation area.
3. Use clear commit messages and explain behavioral impact in the PR description.

## Skill Contribution Checklist

- Keep guidance non-diagnostic and hypothesis-based.
- Avoid permanent labels and deterministic claims.
- Include clear trigger conditions and practical steps.
- Keep wording culturally sensitive and autonomy-supportive.
- Ensure the skill aligns with `CLAUDE.md` and `docs/consultation-protocol.md`.

## Pull Request Expectations

- Describe what changed and why.
- Add a short "before vs after" example when behavior changes.
- Update related docs if workflows or assumptions changed.
